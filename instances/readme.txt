Instances are organized as follows:
The first 5 lines contain the values of the following parameters:

# requests
# SDLs
Tmax
# vehicles
# penalty charged for delivering a request to a SDL
# vehicles usage cost

In the following lines, for each node in the network is reported

<node_id> <x_coord> <y_coord> <TW_start> <TW_end> <service_time> <capacity>

for private locations and for the depot the capacity is always 0 

for shared locations we have positive values of capacity
